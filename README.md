# books-app
Books App fetching data from Google Books API: 
- Search books from Google Books api.
- Bookmark.
- preview book.
- add to plan.
- or reading list.  &nbsp;  &nbsp;
Built using HTML, CSS, javaScript, ReactJs, Redux, React-router, Bootstrap, and Restful API
